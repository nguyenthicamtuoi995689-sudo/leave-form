⚙️ Cấu hình hệ thống
Cần 2 thứ: Google Sheets URL (để lưu data) và EmailJS (để gửi email).
Bước 1 — Google Sheets (Apps Script)
1. Mở Google Sheets → Extensions → Apps Script
2. Xóa code cũ, dán code từ file apps-script.js
3. Nhấn Deploy → New deployment → Web app
4. "Execute as": Me | "Who has access": Anyone
5. Copy URL dán vào ô bên dưới
Bước 2 — EmailJS
1. Đăng ký tại emailjs.com → Add Service → Gmail
2. Tạo Email Template (xem mẫu bên dưới)
3. Lấy Service ID, Template ID, Public Key
